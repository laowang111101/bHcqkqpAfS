# 项目简介

基于YOLOv10的深度学习X光安检危险品图像识别系统，主要用于识别各类行李中的危险品。系统包含功能模块：数据预处理、模型训练、图像识别及结果展示。技术特点为运用了YOLOv10的高效目标检测算法，并提供了Python源码、用户界面、配套数据集及训练代码，助力实现快速、准确的危险品识别，提高安检效率。


## 核心功能
- 数据预处理：对X光安检图像进行清洗、增强等预处理操作
- 模型训练：基于YOLOv10算法进行危险品识别模型的训练
- 图像识别：对输入的X光图像进行危险品自动检测与识别
- 结果展示：以直观方式呈现识别结果，便于安检人员快速判断

## 适用场景
- 机场、车站等公共交通枢纽的行李安检
- 政府、大型活动场所的安全检查
- 物流包裹的违禁品检测
- 边境口岸的入境物品筛查

## 技术特点
- 采用YOLOv10目标检测算法，实现高效准确识别
- 提供完整Python源码，便于二次开发与优化
- 配备图形化用户界面，操作简便直观
- 包含配套数据集与训练代码，支持模型持续改进
